COMMAND(PERINTAH) GIT
0. git config --list = buat ngecek username+email yang terkonfigurasi di git
1. git config --global user.name "username git" = konfigurasi username
2. git config --global user.email youremail@gmail.com = konfigurasi email
3. git clone (link repo) = buat clone(copas) repo
4. git init = inisialisasi repo yang baru
5. git add . = untuk nambahin semua file yang ma di commit
6. git status = buat cek apakah ada file yang di add dan akan di commit
7. git commit -m "keterangan" = buat commit file yang di add ke repo
8. git branch (nama branch) =  buat baru branch
9. git checkout (nama branch) = buat pindah-pindah branch
10. git merge (nama branch baru/yang diubah) = buat gabungin branch lain ke daalam branch master
11. git remote add origin (link repo) = buat remote atau akses repo yang mau kita pakai
12. git pull origin (nama branch) = buat narik perubahan file yang ada di branch
13. git push origin (nama branch) = buat ngirim perubahan ke branch kita
